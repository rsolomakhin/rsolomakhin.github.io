Secure Payment Confirmation (Multi-RP)

This is a test website. Nothing is charged.

This page demonstrates the multi-relying-party feature of Secure Payment Confirmation (SPC).

To use this demo, you need one or more SPC credentials. You can enroll credentials on this page, which will be scoped to this origin (). You can also enroll credentials on another origin, such as stephenmcgruer.com/web-payments-demos/pr/spc/.

After enrolling, copy the base64-encoded credential ID and the RP ID (which is the hostname where the credential was created) into the fields below.

Payment

Enter the details of up to two credentials to use for payment.
