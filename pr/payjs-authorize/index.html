Pay.js with Authorization

This page can be used to test Google Pay's pay.js with payment authorization. The first transaction is simulates a authorization failure. All consequent transactions simulate success.

This demo relies on the UPDATE_PAYMENT_DETAILS Android intent and the UpdatePaymentDetailsIntentFilterInPaymentApp feature flag.
